FastAPI - Add Two Numbers

Concept

This FastAPI program demonstrates how to build a simple web API that adds two numbers using the Union type hint to accept either strings or integers. FastAPI will convert inputs automatically if possible, making the API flexible.

Python Code

from fastapi import FastAPI
import uvicorn
from typing import Union

a = FastAPI()

@a.get("/hi")
def A(a: Union[int, str], b: Union[int, str]):
    return {"result": int(a) + int(b)}

if __name__ == "__main__":
    uvicorn.run(a, host="127.0.0.1", port=5000)

Sample Output

GET /hi?a=5&b=10
→ {"result": 15}

Explanation

  • FastAPI(): Creates a FastAPI application instance.
  • @a.get("/hi"): Defines a GET endpoint at /hi.
  • def A(a: Union[int, str], b: Union[int, str]): Accepts two query parameters, a and b.
  • int(a) + int(b): Converts both inputs to integers before adding.
  • uvicorn.run(...): Starts the FastAPI server on 127.0.0.1:5000.
Tip:

This API is helpful in learning how FastAPI handles query parameters and type conversion using Python’s Union type from the typing module.
